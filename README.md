<img width=400 align=right src="https://github.com/ai-se/ResourceDataDrivenSBSE/raw/master/img/ddsbse17.png">The recent seminar Data-Driven Search-Based Software Engineering (https://goo.gl/f8D3EC) concluded that  (1) the  mining software repositories can be improved using search-based methods; and that (2) search-based methods can be improved using tools from the MSR community. 
  
The resources of this repository are intended to encourage more experimentationin these two directions.
